Beta Description - IDM241

Allison Drake

Triggers

  • On button hover
  • On button click
  • On link hover
  • On link click
  • On exit click

Rules

On button hover:

  • Cursor turns into a pointer
  • Drop shadow forms around button

On button click:

  • Button expands out and down
  • "Share" slides to the top left and decreases in size
  • Exit icon fade in on the top right and three links fade in on the left side as button expands out

On link hover:

  • Underline slides in from left to the right
  • Text fades into bold

On link hover off:

  • Underline slides out from right to left
  • Text fades back to normal

On exit icon click:

  • Exit icon and three links fade out
  • Rectangle contracts back into button
  • "Share" slides back to orginal positon and increases in size

Feedback

Same as rules.

Loops and Modes

No loops or modes.
